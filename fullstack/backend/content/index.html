Login Form

Comment s'authentifier à l'API :

Dans le formulaire "Login Form", entrez les identifiants pour se connecter. Vous recevrez un token.

Pour chaque requête, il faudra mettre ce token dans le header de la requête :

Key: Authorisation

Value: *Le token reçu*

Comment envoyer un mail :

Pour envoyer un mail, il suffit de faire une requête de type POST à l'url http://51.75.126.62:3000/mail

Il est nécessaire d'être authentifier pour envoyer un email !

Structure du JSON :

{
"to": "xxxx@xxx.x",
"subject":"Hello world",
"content":"Si tu lis ce mail, c'est que ca fonctionne !"
}
